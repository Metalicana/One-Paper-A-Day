## What These Notes Include

Notes may include:

- Paper link
- Authors
- Problem the paper addresses
- Method summary
- Applications
- Personal observations

The format is flexible. The goal is to capture the important parts in a way that is easy to revisit later.

## Workflow

1. Read a paper.
2. Add a Markdown note in `Papers`.
3. Write down the key ideas in my own words.
4. Optionally save the PDF next to the note.

## Purpose

This repository is a lightweight knowledge base for papers I want to remember.
